# identificador-oportunidades
Projeto desenvolvido para a disciplina de Aprendizado Profundo, do curso de Pós-graduação em Ciências da Computação [PPGCC] &lt;https://www.ibilce.unesp.br/#!/pos-graduacao/programas-de-pos-graduacao/ciencia-da-computacao/> da Universidade Estadual Paulista "Júlio de Mesquita Filho" [UNESP] ministrada pelo Prof. Dr. Denis Henrique Pinheiro Salvadeo.

# Playwright
Após install a bibliotéca executar no terminal
plawright install
# Para sistemas linux executar também
playwright install-deps

# Ollama
baixar e instalar o software disponivel em: https://ollama.com/
# Executar no terminal
ollama pull <nome_do_modelo_desejado>
# E manter o servidor aberto com o comando
ollama serve
